# Инструкция по развертыванию

## Системные требования
- Node.js v20+
- npm v10+
- Docker и Docker Compose (для контейнеризации)
- Git

## Развертывание локально
1. Клонируйте репозиторий: `git clone <url-репозитория>`
2. Перейдите в папку проекта: `cd web-components-vs-frameworks`
3. Установите зависимости: `npm install`
4. Запустите в режиме разработчика: `npm run dev`
5. Для продакшен-сборки используйте: `npm run build`

## Развертывание через Docker Compose
Данный подход обеспечивает идентичность среды разработки и продакшена.
1. Выполните команду:
   ```bash
   docker compose up --build -d
   ```
2. Флаг `-d` запустит контейнер в фоновом режиме.
3. Остановить контейнер: `docker compose down`

## Публикация на GitHub Pages
Приложение разворачивается автоматически благодаря настроенному CI/CD конвейеру в файле `.github/workflows/ci-cd.yml`.
Триггером для деплоя служит Push-событие в ветку `main`.
Собранная статическая версия публикуется в окружении `github-pages`. 
Для корректной работы путей статики в `vite.config.ts` настроен параметр `base: '/web-components-vs-frameworks/'`.
